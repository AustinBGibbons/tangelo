Information

This demo shows an animation of the sequence of operations performed by the Cross-Cat clustering algorithm developed by Vikash Mansinghka et al. at MIT.

The rows of the table represent the objects (here, animal species) and the columns represent attributes. A Cross-Cat clustering consists of a set of views, or groups of attributes, where each object is assigned to a partition in each view. The number of views and the number of partitions per view are free variables that are detected dynamically based on properties of the data.

Hover over a row to see its corresponding location in other views. Press the play/pause button to start and stop the looping animation. The animation will be smoothest on the latest Google Chrome.
